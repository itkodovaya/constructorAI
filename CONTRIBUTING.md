# Contributing Guide

Спасибо за интерес к проекту Constructor AI Platform!

## Как внести вклад

### 1. Fork проекта

Создайте fork репозитория на GitHub.

### 2. Создайте ветку

```bash
git checkout -b feature/amazing-feature
```

### 3. Внесите изменения

- Следуйте стилю кода проекта
- Добавляйте комментарии к сложным участкам
- Обновляйте документацию при необходимости

### 4. Тестируйте

Убедитесь, что:
- Код компилируется без ошибок
- Все существующие тесты проходят
- Новый функционал работает корректно

### 5. Commit

```bash
git commit -m "Add amazing feature"
```

Используйте понятные сообщения коммитов.

### 6. Push

```bash
git push origin feature/amazing-feature
```

### 7. Создайте Pull Request

Опишите изменения и их цель.

## Стандарты кода

### TypeScript

- Используйте строгую типизацию
- Избегайте `any` где возможно
- Добавляйте JSDoc комментарии для публичных API

### React

- Используйте функциональные компоненты
- Применяйте хуки вместо классов
- Оптимизируйте ре-рендеры

### Стиль

- Используйте Prettier для форматирования
- Следуйте ESLint правилам
- Максимальная длина строки: 100 символов

## Структура коммитов

```
type(scope): subject

body (optional)

footer (optional)
```

Типы:
- `feat`: Новая функция
- `fix`: Исправление бага
- `docs`: Документация
- `style`: Форматирование
- `refactor`: Рефакторинг
- `test`: Тесты
- `chore`: Обслуживание

## Вопросы?

Создайте Issue на GitHub с описанием проблемы или предложения.
