# Tutorials

## Getting Started
1. Launch the constructor.
2. Fill out the brand discovery form.
3. Choose a generated design.
4. Customize your site in the editor.

## Using AI Features
- **Magic Fill**: Automatically generates content for all empty fields.
- **AI Rewrite**: Rewrites a specific block's copy in a different tone.
- **Auto-Evolve**: Launches an A/B test for a block to optimize conversion.
