# Детальный план реализации Phase 6: Autonomous Agents (Ближайшие цели)

Этот документ детализирует задачи первого этапа из Master Roadmap.

## 1. Фреймворк Автономных Агентов (Agent Core) [pending]
- [ ] **Архитектура Message Bus**: Создание внутренней системы обмена сообщениями между агентами через PostgreSQL (Prisma).
- [ ] **Реализация Агента-Планировщика (Orchestrator)**: Агент, который разбивает пользовательский запрос на подзадачи (например: "Создай лендинг для завода" -> "Текст", "Стили", "Блоки", "SEO").
- [ ] **Контекстное Окно Агента**: Система "памяти", позволяющая агенту видеть состояние всего проекта и предыдущие решения.

## 2. Интеллектуальный RAG 2.0 [pending]
- [ ] **Мультимодальная индексация**: Поддержка обработки изображений (OCR) и встраивание их в базу знаний.
- [ ] **Семантический Кеш**: Ускорение повторных запросов за счет хранения похожих ответов в PGVector.
- [ ] **Интерфейс "Библиотекарь"**: Визуальный инструмент для ручной коррекции весов документов в базе знаний.

## 3. GPU Cluster & Ops [pending]
- [ ] **Node Health Monitoring**: Система мониторинга температуры и загрузки локальных видеокарт.
- [ ] **Dynamic Model Loader**: Автоматическая подгрузка/выгрузка моделей (например, смена Llama3 на Mistral) в зависимости от типа задачи.

# Ожидаемые результаты по завершению Phase 6:
1. Пользователь может дать одну команду: *"Сделай полноценный проект бренда X для ниши Y"*, и система через цепочку агентов создаст все ассеты, тексты и структуру автоматически.
2. Система RAG будет работать в 3 раза точнее за счет гибридного поиска (BM25 + Semantic).
3. Платформа будет готова к объединению в кластер из нескольких физических серверов.
